## TUI API接口说明
👉`TUI API使用说明.pdf`👈，详细的描述了TUI内核结构，以及所有API接口的说明和使用demo。

#### TUI框架特性：
* 独立的模块化，支持完美释放资源后退出。
* 独立封装了各个跨平台函数，代码包含`include "tui.h"`就可以。
* 视图、资源、逻辑分离，互不干扰。
* 所有事件都通过回调函数相应。

#### TUI控件特性：
* 支持gif动画
* 支持二维码
* 支持多路声音混播
* 支持各种动画
* 支持多国语言
* 支持点阵字体和矢量字体
